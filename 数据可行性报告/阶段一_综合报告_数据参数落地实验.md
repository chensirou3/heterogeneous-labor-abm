### 1.2 每个参数的实现规格

#### reservation_wage

```
输入：SCE LMS microdata → reservation_wage_dollar (年薪)
处理：
  1. 剔除 <= 0 或明显不合理值
  2. Winsorize at 1st/99th percentile
  3. log_rw = log(reservation_wage_dollar)
  4. 按 employment_status 分组建模
输出：log_rw ∈ R（连续型）
搜索空间：经验分布的 [P5, P95] 区间
```

#### offer_arrival_belief

```
输入：SCE LMS microdata → P(offer in 4 months) ∈ [0, 100]
处理：
  1. belief_01 = raw_value / 100
  2. 按 employment_status 分组建模
输出：belief_01 ∈ [0, 1]（连续型概率）
搜索空间：[0.01, 0.80]（覆盖经验分布的合理范围）
```

#### search_intensity

```
MVP版：
  if status == E: search_intensity = SCE_searched (0 or 1)
  if status == U: search_intensity = CPS_num_active_methods / 9
  if status == N: search_intensity = 0
输出：search_intensity ∈ [0, 1]

增强版（后期）：
  if status == E: 构造 SCE 多变量复合指标
  if status == U: 加权搜索方法（直接联系雇主权重更高）
```

#### participation_propensity

```
输入：CPS PUMD → PEMLR + PRTAGE + PEEDUCA + PESEX
处理：
  1. 定义 age_group: 16-34 / 35-54 / 55+
  2. 定义 edu_group: No College / College+
  3. 定义 sex: Male / Female
  4. LFPR[g] = (E[g] + U[g]) / (E[g] + U[g] + N[g])
输出：participation_propensity[g] ∈ (0, 1)，12组baseline值
搜索空间：以经验LFPR为中心 ± 5个百分点
```

## 输出物 3：Worker Types 初版定义

### 3.1 三类 Worker Types 参数约束

| Worker Type | 标签 | reservation_wage | search_intensity | offer_arrival_belief | participation_propensity | 数据构造规则 |
|-------------|------|-----------------|-----------------|---------------------|------------------------|------------|
| **Type 1** | Active Searcher（积极搜索者） | 中-高（> median） | 高（searched=1 且方法多） | 高（> median） | 高（稳定在LF中） | searched=1 AND offer_arrival_belief > P50 |
| **Type 2** | Low-threshold Acceptor（低门槛接受者） | 低（< P25 或 < median） | 中 | 中 | 高 | reservation_wage < P25 AND participation_propensity > group LFPR |
| **Type 3** | Marginal Worker（边缘参与者） | 不定 | 低（searched=0） | 低（< median） | 低（< group LFPR 或 NILF） | searched=0 AND (offer_arrival_belief < P25 OR status=N) |

### 3.2 类型分配逻辑（MVP 版）

```python
def assign_worker_type(agent):
    """MVP版：基于 searched × offer_arrival_belief × reservation_wage 三维分配"""

    if agent.searched == 1 and agent.offer_arrival_belief > median_oab:
        return 'Type1_ActiveSearcher'

    elif agent.reservation_wage < percentile_25_rw and agent.participation_propensity > agent.group_lfpr:
        return 'Type2_LowThreshold'

    else:
        return 'Type3_Marginal'
```

### 3.3 各类型预期行为特征

| 维度 | Type 1: Active Searcher | Type 2: Low-threshold Acceptor | Type 3: Marginal Worker |
|------|----------------------|---------------------------|----------------------|
| **E→E'（换工作）** | 高——主动搜索更好机会 | 低——已有工作就留下 | 低——不搜索 |
| **U→E（找到工作）** | 高——搜索强+信念高 | 高——门槛低易接受 | 低——不搜索或信念低 |
| **E→U（失业）** | 中——可能主动辞职搜索 | 低——保留工资低不易辞职 | 中-高——可能被解雇或退出 |
| **E→N / U→N（退出LF）** | 低——持续参与 | 低——持续参与 | 高——容易退出 |
| **N→E / N→U（进入LF）** | N/A（很少在N） | N/A | 低——不易重新进入 |

### 3.4 预期类型比例（基于SCE LMS汇总统计推算）

| 指标 | 值 | 推算逻辑 |
|------|---|---------|
| 过去4周搜索者比例 | ~23-28%（SCE LMS） | searched=1 的比例 |
| offer_arrival_belief > median 的搜索者 | ~12-15% | 搜索者中约一半信念较高 |
| → **Type 1 占比估计** | **~12-15%** | |
| reservation_wage < P25 | ~25% | 定义 |
| 其中 participation_propensity 高 | ~18-20% | 排除已在N的 |
| → **Type 2 占比估计** | **~18-20%** | |
| → **Type 3 占比估计** | **~65-70%** | 剩余 |

**关键：Type 3（边缘参与者）占多数。** 这是合理的——大多数就业者不搜索、不主动换工作。Type 3 的内部仍有异质性（稳定就业者 vs 退出风险者），后期可细分。

### 3.5 升级路径

| 版本 | 类型数 | 方法 | 何时使用 |
|------|-------|------|---------|
| **MVP** | 3类 | Rule-based（上述规则） | 第一版，阶段二 |
| **V2** | 3-4类 | K-means / GMM on (rw, oab, si, pp) | 阶段三稳健性 |
| **V3** | 连续 | Mixture distribution + latent class | 后期增强 |

## 输出物 4：MVP 参数分布实现建议

### 4.1 参数分布方案总表

| 参数 | 分布形式 | 分布参数来源 | 搜索空间 | 校准自由度 |
|------|---------|------------|---------|-----------|
| **reservation_wage** | Log-normal（按 E/非E 分组） | SCE LMS 经验分布的 mean 和 sd | μ ∈ [log(30K), log(150K)]; σ ∈ [0.3, 1.2] | 2（μ, σ）× 2组 = 4 |
| **offer_arrival_belief** | Beta（按 E/非E 分组） | SCE LMS 经验分布拟合 | α ∈ [0.5, 5]; β ∈ [2, 20] | 2（α, β）× 2组 = 4 |
| **search_intensity** | 三档离散 | SCE+CPS 经验比例 | p_low, p_mid, p_high（三档比例）| 2（三档比例只有2个自由度）|
| **participation_propensity** | 分组常数（12组 baseline） | CPS 经验 LFPR | 以 LFPR_g 为中心 ± δ, δ ∈ [-0.05, +0.05] | 1（统一偏移量 δ）|

### 4.2 各参数的分布细节

#### A. reservation_wage 分布

```
就业者：log(rw_E) ~ Normal(μ_E, σ_E)
  经验锚点：mean(log(rw)) ≈ log(81K) ≈ 11.30
  经验 sd ≈ 0.6-0.8（需从微观数据确认）
  搜索范围：μ_E ∈ [10.3, 11.9], σ_E ∈ [0.3, 1.2]

非就业者：log(rw_N) ~ Normal(μ_N, σ_N)
  经验锚点：预期低于就业者（文献一致）
  搜索范围：μ_N ∈ [9.9, 11.5], σ_N ∈ [0.3, 1.2]
```

#### B. offer_arrival_belief 分布

```
就业者：oab_E ~ Beta(α_E, β_E)
  经验锚点：mean ≈ 0.18-0.22, 高度右偏
  约束：E[Beta] = α/(α+β) ≈ 0.20

非就业者：oab_N ~ Beta(α_N, β_N)
  经验锚点：预期高于就业者（失业者更期望收到offer）
  约束：E[Beta] = α/(α+β) ≈ 0.25-0.35
```

#### C. search_intensity 分布

```
三档离散分布：
  Low  (si = 0.0): 不搜索 → ~72% (基于SCE: 100% - 28%搜索者)
  Mid  (si = 0.3): 被动搜索 → ~15%
  High (si = 0.8): 积极搜索 → ~13%

CPS增强版（仅失业者）：
  si = num_active_methods / 9  → 连续 [0, 1]
  经验分布：mean ≈ 0.3-0.4, 高度右偏
```

#### D. participation_propensity 分布

```
12组 baseline LFPR（示意值，需从CPS计算精确值）：

            Male              Female
            No College  Col+  No College  Col+
Age 16-34   0.72       0.82   0.63       0.78
Age 35-54   0.85       0.92   0.70       0.80
Age 55+     0.55       0.65   0.40       0.52

校准时统一加偏移量 δ：
  pp_g = LFPR_g + δ,  δ ∈ [-0.05, +0.05]
  clamp to [0.01, 0.99]
```

### 4.3 搜索空间总维度

| 参数 | 自由维度数 | 说明 |
|------|----------|------|
| reservation_wage | 4 | (μ_E, σ_E, μ_N, σ_N) |
| offer_arrival_belief | 4 | (α_E, β_E, α_N, β_N) |
| search_intensity | 2 | (p_mid, p_high); p_low = 1 - p_mid - p_high |

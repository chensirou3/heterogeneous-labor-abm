## Part 4：真实数据版三模型结果

### 总 Loss 比较

| 模型 | Train | Validation | Test | Test/Train | Δ vs A (Test) |
|------|-------|-----------|------|-----------|--------------|
| Model A (Traditional) | 2,815 | 68,119 | **86,968** | 30.89 | baseline |
| Model B (Standard ABM) | 1,326 | 34,884 | **26,303** | 19.84 | -69.8% |
| **Model C (Heterogeneous)** | **1,364** | **29,265** | **17,012** | **12.47** | **-80.4%** |

### 核心发现

1. **Model C 在真实数据测试集上优于 Model A（-80.4%）和 Model B（-35.3%）**
2. **Synthetic 结论在真实数据上成立——幅度有所收缩** (C vs B: synthetic -75.5% → real -35.3%)
3. **过拟合比：** Model C (12.47) 优于 Model B (19.84) 和 Model A (30.89)
4. **Model C 赢得 6/12 目标**，特别是高权重核心目标（UR -99.0%, LFPR -91.0%）

## Part 6：论文级结论

> **在 BLS 真实数据上，异质 ABM 在外样本测试集上相对传统 benchmark 改善 80.4%，相对同质 ABM 改善 35.3%。两大核心机制——reservation wage 异质性（+212%）和匹配市场竞争（+179%）——是改善的主要来源。此结论在合成数据和真实数据上均成立。**

**必须收缩的 Claim：** Declining RW 在低 UR 测试期不稳健；过拟合比从 1.05 升至 12.47。

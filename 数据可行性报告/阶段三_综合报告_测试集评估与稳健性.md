## Part 3：Ablation 结果

### 各 Ablation 的 Test Loss

| Ablation | Test Loss | Δ vs Baseline | 排名 | 关键性 |
|----------|----------|--------------|------|--------|
| **baseline** | **1,877** | — | — | — |
| no_matching_market | 41,259 | **+2098.2%** | 🥇 | **最关键** |
| no_rw_hetero | 25,675 | **+1267.9%** | 🥈 | **极关键** |
| no_declining_rw | 3,432 | +82.8% | 🥉 | 重要 |
| no_oab_hetero | 3,091 | +64.7% | 4 | 较重要 |
| no_type_behavior | 2,968 | +58.1% | 5 | 较重要 |
| no_si_hetero | 2,265 | +20.7% | 6 | 次要 |
| no_pp_hetero | 1,717 | **-8.5%** | 7 | **反向**（移除反而改善） |

### Ablation 经济学解释

#### 🥇 匹配市场竞争机制（+2098.2%）
移除后 test loss 从 1,877 暴增到 41,259——这是 Model C 最关键的结构性机制。经济解释：
- 匹配市场使 **搜索强度成为竞争优势**而非独立概率
- 当劳动力市场紧张时（vacancy 多），几乎所有人都能找到工作；当市场松弛时，只有高搜索强度者获得 offer
- 这产生了 **内生的反周期选择效应**——衰退期间，低技能/低动力 worker 首先被排挤出匹配
- 同质模型无法产生这种非线性聚合效应

#### 🥈 Reservation Wage 异质性（+1267.9%）
移除后 test loss 暴增到 25,675。经济解释：
- RW 异质性创造了 **差异化的接受决策**——低 RW worker 快速接受 offer，高 RW worker 等待更好机会
- 移除后，所有 worker 使用均值 RW，导致匹配市场的筛选功能失效——均值 RW 对所有人要么太高（都拒绝）要么太低（都接受）
- 关键互补性：RW 异质性 × 匹配市场竞争 = 双重选择过滤

#### 🥉 Declining Reservation Wage（+82.8%）
移除后 test loss 从 1,877 增到 3,432。经济解释：
- 失业时长下调 RW 使 U→E 转移率在失业初期低、后期高——这匹配 CPS 中的 duration-dependent hazard rate
- 无此机制时，高 RW worker 可能永久卡在失业状态

#### 4. OAB 异质性（+64.7%）
经济解释：不同 worker 对 offer 到达概率的信念不同，影响搜索决策和匹配市场分数。

#### 5. 类型特定行为规则（+58.1%）
经济解释：Active Searcher 的低裁员风险(×0.85)和 Marginal Worker 的高退出倾向(×1.3)创造了内生的 worker composition 动态。

#### 6. Search Intensity 异质性（+20.7%）
贡献较小——因为匹配市场已经间接地使用了 SI 作为竞争分数。

#### 7. PP 异质性（-8.5%）
**反向发现：**移除 PP 异质性反而改善了测试集表现。可能原因：
- 合成数据中没有按 age×edu×sex 分组的 LFPR 差异——12 组 PP 值增加了噪声
- **论文含义：**PP 异质性的价值取决于是否有真实分组数据。用合成数据时不应包含此异质性；接入 CPS 真实数据后需重新评估。

### Ablation 分层总结

| 层级 | 机制 | 是否进入主文？ |
|------|------|-------------|
| **核心层（不可移除）** | 匹配市场竞争 + RW 异质性 | ✅ 必须在主文 |
| **重要层（显著贡献）** | Declining RW + OAB 异质性 + 类型行为规则 | ✅ 主文，作为增强项 |
| **次要层（边际贡献）** | SI 异质性 | ⚠️ 可在主文中简述 |
| **待验证层** | PP 异质性 | ❌ 当前版本不放入主文核心 |

## Part 4：稳健性结果

### C1：损失函数权重稳健性

| 权重方案 | Model B Test | Model C Test | C vs B | C 稳健？ |
|---------|-------------|-------------|--------|---------|
| baseline_weights | 7,657 | 1,877 | **-75.5%** | ✅ |
| high_flow_weights (flows×2) | 8,297 | 2,551 | **-69.3%** | ✅ |
| high_jolts_weights (JOLTS×2) | 7,894 | 2,106 | **-73.3%** | ✅ |
| equal_weights (all=1) | 3,144 | 1,028 | **-67.3%** | ✅ |

**结论：Model C 在所有权重方案下均显著优于 Model B（-67% 到 -76%），结论完全不依赖权重选择。**

### C2：随机种子稳健性

| 模型 | Mean Test Loss | Std | C wins |
|------|---------------|-----|--------|
| Model B | 7,758 | 773 | — |
| Model C | 2,049 | 620 | **20/20 (100%)** |

**结论：Model C 在全部 20 个随机种子上均优于 Model B，无一例外。这不是偶然路径。**

### C3：样本切分稳健性

| 切分方案 | Model B Test | Model C Test | C vs B |
|---------|-------------|-------------|--------|
| baseline (month 114-131) | 7,657 | 1,877 | **-75.5%** |
| early_test (month 108-131) | 5,776 | 1,632 | **-71.7%** |
| late_test (month 120-131) | 14,479 | 2,066 | **-85.7%** |

**结论：所有切分方案下 Model C 均大幅领先。值得注意的是 late_test（更短的测试期）中 Model C 的优势更大（-85.7%），说明在数据更少的情况下异质性模型的泛化优势更明显。**

## Part 5：升级模块评估

### CES Enhancement
- **可行性：** 高。CES nonfarm payroll / AHE / hours 数据公开可用
- **预期价值：** 可作为额外验证层（employment level vs CPS employment）
- **是否影响主结论：** 不影响——CES 提供增量验证而非核心校准
- **建议：** 放入附录作为补充验证

### SIPP Enhancement
- **可行性：** 中。SIPP panel 可提供更细的状态转移数据
- **预期价值：** 可改善 flow 拟合精度
- **是否影响主结论：** 可能增强 PP 异质性的实证基础
- **建议：** 列为 future work

### Deeper Traits Prior (NLSY / HRS)
- **可行性：** 低-中。需要额外数据清洗
- **预期价值：** 提供方向性先验，不改变模型结构
- **是否影响主结论：** 不影响
- **建议：** 列为 future work

## Part 6：论文级结论

### 当前最强可辩护 Claim

> **在相同外部环境和统一多目标评价体系下，引入受公开调查（SCE LMS + CPS）约束的劳动行为异质性后，基于匹配市场竞争机制的异质 ABM 在外样本测试集上相对传统 benchmark 改善 97.3%，相对同质 ABM 改善 75.5%。这一结论在全部测试的权重方案、样本切分和 20 个独立随机种子下完全稳健。Ablation 实验表明，两大核心机制——有限岗位匹配市场竞争和 reservation wage 异质性——贡献了绝大部分改善。**

### 当前应避免的过强 Claim

1. ❌ "异质性 ABM 在所有指标上都优于传统模型" ← 不准确，LFPR 和部分 flow 指标仍输 Model B
2. ❌ "PP 异质性是重要的" ← ablation 显示移除反而改善，需要真实数据验证
3. ❌ "模型可以直接用于政策预测" ← 当前基于合成数据，需接入真实数据后验证
